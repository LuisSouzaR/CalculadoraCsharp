### Calculadora Simples em CSharp

> Projeto de uma calculadora Simples (Com os 4 operadores) Feita na Linguagem CSharp

A calculadora conta com botões números, os 4 operadores principais, Adição, Subtração, Multiplicação e Divisão. Além de contar com botão de backspace, ClearSpace, e Clear All, e operação de porcentagem, suportando por fim números decimais.
